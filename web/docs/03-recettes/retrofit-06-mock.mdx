# HTTP 6 - Mock de serveur

Il se peut qu'on doivent programmer une application qui parle avec un serveur ...
mais un serveur qui n'existe pas encore.

On va alors vouloir **simuler** le serveur pour pouvoir continuer à travailler sur l'application.

Retrofit permet de coder en dur les réponses pour commencer le travail avec un **faux** serveur.
